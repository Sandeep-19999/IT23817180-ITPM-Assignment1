## Project Description

This project contains automated test cases for evaluating the **Chat Sinhala transliteration function** available in the Pixelssuite Chat Translator web application.

The main purpose of this assignment is to identify situations where the system fails to correctly convert **chat-style Singlish input** into the expected Sinhala output.

The testing scope is limited to the Chat Sinhala transliteration feature only. This project does not test backend APIs, performance, scalability, security, or the Standard Sinhala transliteration function.
